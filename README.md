## Course Information

**Course:** CSE3433 Software Architecture  
**Faculty:** Faculty of Computer Science and Mathematics

**Semester:** Semester II 2025/2026

## Project Overview

The Cloud-Native Learning Management System (LMS) is designed to provide a scalable, modular, and reliable learning platform for students, lecturers, and administrators.

The system adopts a cloud-native architecture approach using API-based communication between frontend and backend services to improve scalability, maintainability, and performance.
